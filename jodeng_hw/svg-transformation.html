Transformation

SVG이미지에서는 생성된 Shapes(circle, rect, line, poligon, text etc.,)들의 변형시키는 것이 가능하다. (ex. move, scale, rotate)
수직 또는 대각선 텍스트를 표현하는데도 편리한 방법을 제공한다.

transform 속성 - 적용 범위

  • <circle><rect><text><line>와 같은 Shape들
  • <g> : 효과적으로 한번에 전체 그룹에 변형
  • 패턴을 칠하고 gradient를 변형시키는 것이 가능

Functions

  • translate()
  • rotate()
  • scale()
  • skew()
  • matrix()

Translate

- 이동
- translate(50, 25)
-> x축으로 50, y축으로 25 만큼 이동해라. <rect x="20" y="20" width="50" height="50" style="fill: #cc3333"/> <rect x="20" y="20" width="50" height="50" style="fill: #3333cc" transform="translate(75,25)"/>

Rotate

- 회전
- rotate(15)
-> (0,0)을 기준으로 15도 회전해라.
- 시계 방향으로 회전 (0 ~ 360)
- 반시계 방향으로 회전 (-40) : '-'음수 표시
<rect x="20" y="20" width="40" height="40" style="stroke: #3333cc; fill:none;" /> <rect x="20" y="20" width="40" height="40" style="fill: #3333cc" transform="rotate(15)" //rotate(15, 20, 20) rotate(15, 40, 40) />

Scale

- scale(2) // x축, y축 2배
- scale(2, 3) // x축 2배, y축 3배
<rect x="10" y="10" width="20" height="30" style="stroke: #3333cc; fill:none;"/> <rect x="10" y="10" width="20" height="30" style="stroke: #000000; fill:none;" transform="scale(2)"/> <rect x="10" y="10" width="20" height="30" style="stroke: #0000ff; fill:none;" transform="scale(2,3)"/>

Mirror (거울 효과)

<path d="M20,20 l20,20 l0,20 l-20,20 Z" style="stroke: #3333cc; fill:#00ff00;"/> <path d="M20,20 l20,20 l0,20 l-20,20 Z" style="stroke: #000000; fill:#ffff00;" transform="translate(100, 0) scale(-1, 1) "/>

Skew

- 비틀기
- skewX(), skewY()함수를 이용해 x축, y축으로 비튼(?) 기울인다.
- 실제로, 특정 각도를 지정해주면 축을 기울인다. <rect x="10" y="10" width="20" height="30" style="stroke: #3333cc; fill:none;"/> <!-- skewX() --> <rect x="10" y="10" width="20" height="30" style="stroke: #3333cc; fill:#ba6098;" transform="skewX(85)"/> <rect x="50" y="10" width="20" height="30" style="stroke: #000000; fill:#de97c3;" transform="skewX(10)"/> <rect x="100" y="10" width="20" height="30" style="stroke: #000000; fill:#de97c3;" transform="skewX(45)"/> <rect x="150" y="10" width="20" height="30" style="stroke: #000000; fill:#de97c3;" transform="skewX(60)"/> <!-- skewY() --> <rect x="10" y="10" width="20" height="30" style="stroke: #3333cc; fill:#47c27c;" transform="skewY(85)"/> <rect x="50" y="10" width="20" height="30" style="stroke: #000000; fill:#82e4b2;" transform="skewY(60)"/> <rect x="100" y="10" width="20" height="30" style="stroke: #000000; fill:#82e4b2;" transform="skewY(45)"/> <rect x="150" y="10" width="20" height="30" style="stroke: #000000; fill:#82e4b2;" transform="skewY(10)"/>
