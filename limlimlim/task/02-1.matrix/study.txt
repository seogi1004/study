
일차변환

점이 변환되어 새롭게 만들어지 점을 나타내는 관계식이
일차식인 경우

일차식 : 상수항 없음 / x, y 일차식


변환행렬 관계식
f( P )  = P`
AP = P`
A = 변환행렬


일차 변환 f를 나타내는 행렬 A
A=
a,b
c,d

a,b     1   a
c,d  x  0 = c

a,b     0   b
c,d  x  1 = d
