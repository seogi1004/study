Animation(SMIL)

기본적인 SMIL(선언적 애니메이션), 즉 element 선언을 이용한 animation 방법이다.

Specific attributes

  • attributeType : 타입정의
  • attributeName : 대상으로 하는 attribute
  • from : 시작점
  • to : 끝점
  • dur : 시간
  • repeatCount : 반복

Example

basic animate multiple animate animateTransfrom(rotate) animateTransfrom(scale) animateMotion
basic animate - click circle!
multiple animate - click circle!
animateTransfrom(rotate)
animateTransfrom(scale)
